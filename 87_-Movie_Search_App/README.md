# 🎬 Movie Search App

A dynamic movie search interface that fetches movie data from a public API. This project highlights API usage, search functionality, and responsive UI design.

---

## ✨ Features

- Search movies by title  
- Displays posters, ratings, and descriptions  
- Responsive layout  
- Error handling for missing results  

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Fetch API)**  
- **OMDb API** (or whichever API you used)
