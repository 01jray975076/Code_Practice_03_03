📘 Personal Finance Tracker
A clean, responsive, browser‑based finance tracker that helps you log income and expenses, visualize spending patterns, and monitor your financial balance — all stored locally in your browser.
This project uses HTML, CSS, vanilla JavaScript, and Chart.js to deliver a lightweight, fast, and user‑friendly experience.
✨ Features
💸 Add Transactions
Track income and expenses
Choose from preset categories (Food, Bills, Fun, Other)
Enter amount and date
Transactions are saved automatically to LocalStorage
📊 Interactive Charts
Powered by Chart.js:
Monthly Income vs Expense bar chart
Category Spending Breakdown bar chart
Charts update instantly when transactions change
🧮 Summary Dashboard
Total Income
Total Expense
Current Balance
Auto‑updates with every transaction
🗂 Transaction List
Sorted by most recent
Shows category, type, date, and amount
Delete individual transactions
Smooth UI with clean formatting
🌗 Light / Dark Mode
Toggle theme with one click
Theme preference saved in LocalStorage
Fully styled for both modes
🛠️ Technologies Used
Technology	Purpose
HTML5	Structure and layout
CSS3	Styling, theming, responsive design
JavaScript (ES6)	App logic, DOM updates, LocalStorage
Chart.js	Data visualization
LocalStorage API	Persistent data storage
