# 🎨 Color Palette Generator

A simple and interactive Color Palette Generator built with **HTML**, **CSS**, and **JavaScript**.  
Click the **Generate Palette** button to instantly create five new random colors.  
Each color card displays its hex value and includes a **copy-to-clipboard** button for quick use in your projects.

---
## 🚀 Features

- Generates **five random colors** with a single click  
- Displays each color’s **hex value**  
- **Copy** any hex value by clicking the copy icon or the color block  
- Smooth UI feedback when a color is copied  
- Clean, responsive layout

---
## 🧩 How It Works

- `generateRandomColor()` creates a random hex color string  
- `generatePalette()` builds an array of five colors  
- `updatePaletteDisplay()` updates the UI with new colors and hex values  
- Event delegation handles clicks on:
  - `.copy-btn` → copies the hex value  
  - `.color` → also copies the hex value

## ▶️ Getting Started

1. Clone or download the project  
2. Open `index.html` in your browser  
3. Click **Generate Palette**  
4. Click any color or copy icon to copy the hex code

No build tools or dependencies required — it’s pure front‑end.
