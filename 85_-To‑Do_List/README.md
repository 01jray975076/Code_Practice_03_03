# 📝 To‑Do List App

A clean and simple To‑Do List application built with HTML, CSS, and JavaScript. This project focuses on DOM manipulation, user interaction, and basic data handling in the browser.

---

## ✨ Features

- Add new tasks  
- Mark tasks as completed  
- Delete tasks  
- Clean, minimal UI  
- Fully client‑side — no backend required  

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**
