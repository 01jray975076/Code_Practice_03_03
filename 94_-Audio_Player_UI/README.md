# 🎧 Audio Player UI

A sleek, minimal, and responsive audio player interface built using HTML, CSS, and JavaScript. This project focuses on front‑end UI design and interactive media controls, offering a clean and modern user experience.

---

## ✨ Features

- **Custom UI Design** — Stylish layout with modern visual elements  
- **Play / Pause Controls** — Fully functional audio playback  
- **Progress Bar** — Visual timeline that updates as the audio plays  
- **Responsive Layout** — Works across different screen sizes  
- **Lightweight & Fast** — Pure front‑end implementation with no dependencies  

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**
