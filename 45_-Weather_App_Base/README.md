🌦️ Weather App — Front-End + API
A clean, modern weather dashboard using a real weather API.
📌 Overview
This project teaches you how to work with real APIs, handle asynchronous data, and update the UI dynamically. It’s one of the most valuable skills for front-end roles.
You’ll build a weather dashboard that shows:
Current temperature
Weather conditions
Location name
Weather icon
Optional: 5‑day forecast
⭐ Features
Search for any city
Fetch real weather data from an API
Display temperature, humidity, wind, and conditions
Weather icons
Error handling (invalid city, network issues)
Optional: Save recent searches
Optional: Dark/light theme
🧠 Skills You’ll Practice
Fetch API
JSON parsing
Async/await
DOM manipulation
Error handling
UI state updates
Responsive layout

3. Build the UI
We’ll create:
Search bar
Weather card
Icon + temperature
Details section
4. Connect to the API
Use fetch() with async/await.
5. Display the data
Update the UI with real weather info.
