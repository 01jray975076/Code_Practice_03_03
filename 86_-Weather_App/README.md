# 🌦️ Weather App

A simple and responsive Weather App that fetches real‑time weather data using a public API. This project demonstrates API integration, async JavaScript, and dynamic UI updates.

---

## ✨ Features

- Search weather by city  
- Displays temperature, humidity, and conditions  
- Clean and responsive UI  
- Error handling for invalid locations  

---

## 🛠️ Tech Stack

- **HTML5**  
- **CSS3**  
- **JavaScript (Fetch API)**  
- **OpenWeatherMap API** (or whichever API you used)
